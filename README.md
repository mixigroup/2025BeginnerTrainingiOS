# 株式会社MIXI 新卒向けiOS研修リポジトリ 2024年版

## 概要

簡易的なGitHubクライアントアプリの開発を通して、SwiftUIを使ったiOSアプリ開発の基礎知識と実務スキルを身に付けるための研修です。iOSアプリ開発未経験者でもSwiftの基礎から学んでいけるように構成されています。各セッションはそれぞれブランチに分かれていて、セッションごとの差分を確認することができます。

## 環境

- Xcode 16.2
- Swift 6.0.3
